1. 입력된 핀수를 심볼로 변경 (#1)
    1. 10 -> Strike -> X
    1. 8 -> Pitching -> 8
        * 2 -> Spare -> 8|/
    1. 8 -> Pitching -> 8
        * 1 -> Open -> 8|1
    1. 0 -> Pitching -> -
        *  0 -> Open -> -|-

1. 입력된 핀수 유효성 체크 (#2)

1. 원시값 포장하기 (#3)

    * Strike
    * Spare
    * Open
    * 프레임 종료 상태

1. 두 개의 프레임을 만들자. (#5)

1. final frame 개념을 고민하자. (#6)

1. 10 프레임 데이터를 만들자 (#7)

1. 조립을 하자. (#8)
    * 다음 프레임 번호
    * 출력하자.
